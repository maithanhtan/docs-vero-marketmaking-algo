| Thuật ngữ | Ý nghĩa ngắn |
|-----------|----------------|
| **CW** | Chứng quyền. |
| **Mã cơ sở / Underlying** | Tài sản cơ sở (vd: CP) mà CW tham chiếu. |
| **Delta (ròng)** | Độ nhạy giá trị CW với giá cơ sở; tính KL cần hedge. |
| **Notional** | Giá trị danh nghĩa (giá × KL). |
| **Hedge** | Mua/bán cơ sở (hoặc công cụ khác) để bù rủi ro CW. |
| **Độ phủ (Coverage)** | % vị thế đã hedge / lượng cần hedge. |
| **Residual** | Phần dư/thiếu sau so sánh yêu cầu và vị thế. |
| **VALID / WARN / INVALID** | Hedger: hợp lệ / cảnh báo / lỗi. |
| **BEST / LAST / MTL** | Chế độ lệnh hedge: BEST = giá tốt nhất sổ lệnh, LAST = giá khớp cuối, MTL = mid/reference. |
| **bps** | Phần vạn (0,01%). Spread, khoảng giá. |

- Xem danh sách thuật toán MM; chọn một để xem chi tiết, cấu hình, lệnh, nhật ký, giao dịch.
- **Bắt đầu** / **Tạm dừng** / **Dừng** / **Sửa** / **Xóa**.

## Danh sách (cột trái)

- Thu gọn/mở: nút mũi tên.
- Chọn: bấm dòng (mã CK, #xxxx, trạng thái). **Trạng thái:** Running (xanh), Paused (vàng), Stopped (xám).
- Chưa có algo: dùng **Tạo mới** hoặc **Nhập**.

## Khung chi tiết (phải)

- **Header:** Mã CK, **Mode** (Sideway / Bullish / Bearish / Following), trạng thái.
- **Config:** Tham số đang chạy (chỉ xem; đổi qua **Sửa**).
- **Tab con:** **Orders** (lệnh, phân trang), **Logs** (nhật ký), **Trades** (giao dịch đã khớp).
- **P&L / Volume** (nếu có).
- **Nút:** Bắt đầu, Tạm dừng, Dừng, Sửa, Xóa.

### Ý nghĩa

| Block | Ý nghĩa |
|-------|---------|
| **Mode** | Sideway / Bullish / Bearish / Following (theo giá underlying). |
| **Orders** | Order ID, Side, Qty, Price, Status, Time. |
| **Logs** | INFO / WARN / ERROR. |
| **Trades** | Giao dịch đã khớp, đối chiếu P&L. |

Xem **chi tiết một mã cơ sở:** bảng hedge từng CW, lệnh hedge, nhật ký hedger. **Tải báo cáo Excel.** Điều khiển hedger (Bắt đầu / Tiếp tục / Giữ Hedge).

## Thao tác

- **Chọn mã cơ sở:** Dropdown góc phải trên → toàn bộ số liệu theo mã đó.
- **Báo cáo:** Loại — **Báo Cáo Định Kỳ Tháng TCPH** / **Báo Cáo Phòng Ngừa Rủi Ro Hàng Ngày**. Phạm vi — **Mã đang chọn** / **Tất cả mã cơ sở** → **Tải xuống**.
- **Cài đặt Hedge:** Mở cấu hình hedge cho algo **đang chọn** ở tab Điều khiển.
- **Bắt đầu / Tiếp tục Hedge / Giữ Hedge:** Cho algo đang chọn. Chưa chọn algo → nút Bắt đầu vô hiệu.
- **Tự động làm mới**, **Làm mới** (giống tab Tổng hợp).

### Bảng CW

- Lọc: **Tất cả**, **Thiếu hedge**, **Hedge dư**, **Hedger lỗi**. Sắp xếp cột. Phân trang (vd: 25/trang).

### Thẻ tóm tắt

| Trường | Ý nghĩa |
|--------|---------|
| Tổng delta ròng, Notional CW | Mã đang chọn. |
| Tổng KL hedge yêu cầu / vị thế / trên TT | |
| Độ phủ tổng thể | % đã hedge. |
| Trạng thái hedger | Số CW VALID / WARN / INVALID. |
| Tổng KL residual, Số warrant thiếu hedge, hedger lỗi | |
| Warrant thiếu hedge lớn nhất | Mã CW và giá trị. |

### Bảng hedge theo CW

| Cột | Ý nghĩa |
|-----|---------|
| Chứng quyền, Trạng thái hedger, Thông điệp trạng thái | |
| Delta ròng, KL hedge yêu cầu/vị thế/residual | |
| KL thiếu/dư, Notional thiếu/dư, Độ phủ | |

### Bảng Lệnh Hedge

Thời gian, Chứng quyền, Chiều, KL, Giá, Trượt giá, Trạng thái (FILLED / PARTIAL / REJECTED). Có thể có: Tổng KL đã hedge, Giá hedge TB, Trượt giá TB, Tỷ lệ filled, Số partial/reject.

### Bảng Nhật ký Hedger

Thời gian, Mức log (INFO/WARN/ERROR), Hành động, Chứng quyền, Thông điệp.
